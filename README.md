# Бази Даних: Електронний каталог наукових досліджень

Репозиторій містить матеріали лабораторних робіт з дисципліни "Бази Даних".
СУБД: PostgreSQL | Інструментарій: pgAdmin 4

## Лабораторна робота №1
Тема: Проектування бази даних та ознайомлення з базовими операціями PostgreSQL.
Мета: Розробити концептуальну модель (ERD), нормалізувати її до 3НФ та реалізувати фізичну схему бази даних.

### Файли роботи (lab1)
| lab1/lab1.sql | Головний файл. SQL-скрипт для створення всіх таблиць та наповнення їх даними. |
| lab1/lab.pgerd | Файл проекту ER-діаграми (відкривається через pgAdmin 4). |
| .docx | Повний звіт з описом предметної області та нормалізації. |
| BDLAB12_2.png | Графічна схема бази даних (ER-діаграма). |
| BDLAB12_1.png | Скріншот створених таблиць у pgAdmin. |

### Як розгорнути
1. Створити нову базу даних у pgAdmin 4.
2. Відкрити Query Tool.
3. Завантажити та запустити код з файлу lab1/lab1.sql.
